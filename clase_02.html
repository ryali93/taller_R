CONDICIONALES

La toma de decisiones es una parte importante de la programaci√≥n. Esto se puede lograr en la programaci√≥n R usando la nomenclatura condicional if else.

If

if (expresion) {
  declaracion
}
  • Si expresion es TRUE, la instrucci√≥n se ejecuta. Pero si es FALSO, no pasa nada.
  • Aqu√≠, expresion puede ser un vector l√≥gico o num√©rico, pero solo se tiene en cuenta el primer elemento.
  • En el caso del vector num√©rico, cero se toma como FALSO, el resto como VERDADERO.

If‚Ä¶Else

if (expresion) {
    declaracion1
  }else{
    declaracion2
}

La parte else es opcional y solo se eval√∫a si expresion es FALSE. Es importante tener en cuenta que else debe estar en la misma l√≠nea que las llaves de cierre de la instrucci√≥n if.

Condicionales

ITERADORES, BUCLES O LOOPS

Los bucles son una de las estructuras fundamentales en la programaci√≥n. Los bucles permiten iterar sobre cada elemento en una secuencia, uno a la vez, y hacer algo con esos elementos.

FOR

Un bucle for es muy valioso cuando necesitamos iterar sobre una lista de elementos o un rango de n√∫meros. Loop se puede usar para iterar sobre una lista, marco de datos, vector, matriz o cualquier otro objeto. Los tirantes y el corchete son obligatorios.

for (valor in secuencia)
  {
    cuerpo
  }

# Creando un vector de frutas

 print(i)
